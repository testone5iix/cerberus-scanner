# Cerberus Scanner

## نظرة عامة
Cerberus Scanner هي أداة أمنية متقدمة مخصصة لفريق الدفاع السيبراني (Blue Team).  
تعمل على فحص النظام بعمق لاكتشاف:
- حقن الأكواد الخبيثة في ذاكرة العمليات.
- التهديدات في وحدات النواة (Rootkits / Kernel Modules).
- العمليات المخفية (Hidden Processes).
- التحقق من التوقيع الرقمي للسواقات.
- تسجيل نتائج مفصلة مع دعم إخراج JSON لتكامل أدوات SIEM.

هذه الأداة مخصصة لأنظمة Windows وتستخدم وظائف منخفضة المستوى (NTAPI و Win32) للوصول إلى معلومات حساسة ودقيقة.

---

## الاستخدام المستهدف
- فرق الدفاع السيبراني في المؤسسات لمراقبة بيئات Windows بشكل مستمر.
- فرق مراكز العمليات الأمنية (SOC) لتحليل الأحداث بعد وقوع الحوادث (Incident Response).
- الباحثون والمهندسون الأمنيون لاختبار الأنظمة وضمان خلوها من التهديدات العميقة.

---

## المزايا
- فحص شامل故故故故 لذاكرة جميع العمليات مع الكشف عن الأنماط المميزة لحقن الشيفرات (shellcode / reflective DLL).
- تحليل وحدات النواة للتأكد من عدم تعديلها من قبل rootkits أو تهديدات مشابهة، من خلال مقارنة محتوى الذاكرة مع الملفات على القرص.
- اكتشاف العمليات المخفية التي قد تستخدم تقنيات متقدمة للتخفي.
- إمكانية الإنهاء التلقائي للعمليات المريبة (مع إمكانية تعطيل ذلك باستخدام `--no-kill`).
- وضع التشغيل التجريبي الآمن (`--dry-run`) لتسجيل الإجراءات دون تنفيذها فعليًا.
- إخراج سجل JSON متكامل لتسهيل الربط مع منصات SIEM أو التحليل اليدوي لاحقًا.
